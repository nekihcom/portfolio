# 概要

このポートフォリオサイトは、私のスキルや経験、制作した作品を紹介するためのウェブサイトです。
Next.jsとTypeScriptを使用して開発し、Tailwind CSSでスタイリングを行いました。

# 主な機能

- レスポンシブデザイン
- 作品の詳細表示
- スキルセットの表示
- コンタクトフォーム

# 技術スタック

- **フロントエンド**
  - Next.js
  - TypeScript
  - Tailwind CSS
  - Framer Motion

- **デプロイ**
  - Vercel

# 開発のポイント

1. **パフォーマンス最適化**

2. **アクセシビリティ**
   - セマンティックHTML
   - キーボードナビゲーション
   - スクリーンリーダー対応

3. **SEO対策**
   - メタタグの最適化
   - 構造化データの実装
   - サイトマップの生成
