# コミットメッセージのルール

## 基本形式
```
<絵文字>: <変更理由や箇所を簡潔にまとめた文章>
```

## 絵文字の使用ルール

### 機能追加・変更
- ✨: 新機能の追加
- 🔄: 機能の更新・変更
- 🎨: UI/UXの改善
- 💄: スタイルの変更
- 🚀: パフォーマンスの改善

### バグ修正
- 🐛: バグ修正
- 🔧: 設定ファイルの変更
- 🚑: 緊急の修正

### ドキュメント
- 📝: ドキュメントの追加・更新
- 💡: コメントの追加・更新

### リファクタリング
- ♻️: コードのリファクタリング
- 🔨: コードの修正
- ⚡️: コードの最適化

### その他
- 🎉: 初回コミット
- 🔥: ファイルの削除
- 📦: 依存関係の追加・更新
- 👷: CI関連の変更
- 🔍: コードの検証・テスト
- 🚀: デプロイ関連の変更

## 例
```
✨: プロフィールセクションにSNSリンクを追加
🎨: お問い合わせフォームのレイアウトを改善
🐛: モバイル表示時のレイアウト崩れを修正
